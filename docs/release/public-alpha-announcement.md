## Short announcement

Chats are temporary. Artifacts should be durable.

I’m opening the public alpha for **Imbas Artifact Vault**, the first wedge of Imbas OS: a local-first desktop vault for AI-generated HTML apps, dashboards, reports, simulations, calculators, and mini-tools.

You can paste or import generated HTML, replay it safely in a sandbox, add notes/metadata/provenance, snapshot versions, search later, and export an AI context package for the next model pass.

Repo: https://github.com/ObiJuanDeanobi/imbas-os

## Longer announcement

LLM output is moving beyond plain text and Markdown into interactive HTML: dashboards, visual explainers, slide decks, simulations, calculators, compliance/evidence views, and single-file mini-tools.

Those artifacts are useful, but they usually get lost in chat history, downloads folders, or one-off project directories.

**Imbas Artifact Vault** gives them a durable local home:

- save/import generated HTML;
- replay it through a hostile-document sandbox;
- keep metadata, notes, provenance, trust level, and snapshots;
- search your artifacts later;
- export an AI context package when you want another model to continue the work.

This is the first public alpha wedge of the broader Imbas OS roadmap: a local-first workbench for artifacts, knowledge, memory, agent runs, approvals, and mobile capture. The public alpha is intentionally narrow: make generated HTML artifacts durable, inspectable, replayable, and reusable first.

Repo: https://github.com/ObiJuanDeanobi/imbas-os

## Suggested GitHub release blurb

This public alpha introduces Imbas Artifact Vault, the first focused wedge of Imbas OS: a local-first Electron desktop vault for AI-generated HTML artifacts.

It can save/import generated HTML, replay artifacts in a sandboxed `artifact://` viewer, preserve metadata/notes/provenance, create/restore snapshots, search artifacts and Markdown pages, and export AI context packages for follow-up model work.

This is an alpha. It is not yet a signed production installer or the full Imbas OS 1.0 agent workbench.
