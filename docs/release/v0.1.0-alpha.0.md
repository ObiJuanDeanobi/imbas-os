# v0.1.0-alpha.0 — Imbas Artifact Vault public alpha

## What this alpha is

This alpha opens the first public wedge of Imbas OS: **Imbas Artifact Vault**, a local-first desktop vault for AI-generated HTML artifacts.

The goal is narrow and practical: make generated HTML artifacts durable, inspectable, replayable, searchable, versioned, and reusable with AI context.

## Highlights

- Electron + React + TypeScript desktop shell.
- Local artifact vault initialization.
- Paste/import generated HTML and replay it through a sandboxed `artifact://` viewer.
- Artifact-origin network requests blocked by default.
- Artifact bundles with metadata, notes, provenance, trust level, hashes, and snapshots.
- Snapshot creation and reversible restore flow.
- Search across artifact titles, tags, notes, prompts, and visible HTML.
- Vault-owned Markdown pages and read-only Markdown/wiki bridge.
- Graph/backlink foundations across artifacts and Markdown pages.
- AI context package export for follow-up model work.
- Demo vault with realistic artifact examples.
- Security smoke tests for generated HTML replay boundaries.
- Current app-shell screenshot and launch assets.

## What is intentionally not stable yet

- Signed/notarized production installers.
- Hosted/cloud sync service.
- Full Imbas OS 1.0 agent operating layer.
- Public-stable Android companion, Memsocket, Conduit, Runledger, Lorekeeper, Sanctum, or live agent dispatch.
- Production backup/restore/delete/forget workflow.

## Verification for the release candidate

```bash
npm run docs:llms:check
npm run check
npm run package:dev
```

```bash
git grep -n -E "tail[[:alnum:]]+\.ts\.net|100\.(6[4-9]|[78][0-9]|9[0-9]|1[01][0-9]|12[0-7])\.|api[_-]?key|secret|token|password|BEGIN .*PRIVATE|PRIVATE KEY" -- . ':!package-lock.json'
```

## Release assets

Prepared release-candidate assets:

- `imbas-os-v0.1.0-alpha.0-dev-preview.tgz` — verified local dev-preview package/restore artifact.
- `imbas-os-v0.1.0-alpha.0-dev-preview.tgz.sha256` — checksum for the dev-preview package.
- `imbas-os-v0.1.0-alpha.0-linux-x64.tar.gz` — unsigned Linux x64 Electron app bundle, when produced by the release-prep pass.
- `imbas-os-v0.1.0-alpha.0-linux-x64.tar.gz.sha256` — checksum for the unsigned Linux x64 bundle.

The Linux bundle should pass Electron smoke and security smoke locally with `--no-sandbox` before attachment. It is unsigned and not a production installer. Package-registry publishing, signed installers, notarization, and hosted/binary distribution beyond GitHub release assets require separate explicit release decisions.
