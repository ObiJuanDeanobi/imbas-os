# ImbasOS Asset Pack

Generated from the latest blue/gold Celtic-inspired ImbasOS concept board.

## Contents

- `logos/` - primary lockups, wordmark crops, icon mark crops, monochrome variants
- `icons/` - app icon options and standalone mark alternatives
- `marketing/` - hero/banner/promo/footer crops
- `ui-mockups/` - desktop dashboard mockup crop
- `source/` - full source board and palette swatch

Notes:
- These are cropped raster PNG assets from the concept board, not final vector artwork.
- Some small text in the generated mockups may be imperfect; use these as visual direction and placeholders.
- For production, recreate the selected logo mark/wordmark as SVG or ask a designer/agent to vectorize the preferred mark.
