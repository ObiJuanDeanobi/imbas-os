# ImbasOS HQ Asset Pack

This pack includes high-resolution PNG concept assets plus clean SVG vector recreations for repo, README, and product use.

## Folders

- `png/hq/` — generated high-resolution raster concepts.
- `png/upscaled_2x/` — 2x upscaled PNGs for larger mockups and banners.
- `svg/` — clean editable SVG vector assets inspired by the selected blue/gold Celtic ImbasOS direction.
- `docs/brand_tokens.json` — palette and tagline references.

## Notes

The SVG files are clean vector recreations, not pixel-perfect traces of the generated raster images. They are better for codebases, icons, README usage, and future refinement. The PNGs preserve the richer 3D rendered style from the generated concepts.

Recommended public naming:

- Product umbrella: `ImbasOS`
- First app/module: `Artifact Vault`
- Tagline: `Local-first. Agent-ready.`
